# Foxar CLI

Command-line interface for FoxarKit - build WebAssembly apps for AR education.

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/FoxarXR/foxarkit-cli/main/install.sh | bash
```

## Requirements

- macOS (Apple Silicon)

## Usage

```bash
foxar --help
foxar build
```

## Manual Installation

```bash
# Download binary
curl -fsSL https://raw.githubusercontent.com/FoxarXR/foxarkit-cli/main/bin/foxar -o foxar
chmod +x foxar
sudo mv foxar /usr/local/bin/
```

## Uninstall

```bash
sudo rm /usr/local/bin/foxar
```
